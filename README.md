#### Calculator_for_Android

# 계산기 프로그램

### 1) 주요기능
1. 기초 연산
    - 사칙연산
    - 나머지 연산(Modular)
    - 비트 연산 (AND, OR, NOT, XOR)
    - 다항식 연산

2. 입력
    - 키보드 입력
    - 버튼 입력
 
3. 츌력
    - 화면 출력
    - 파일 출력

4. 결과 저장(History)
    - 최근 연산 목록 리스트 저장
    - History는 최대 50개
